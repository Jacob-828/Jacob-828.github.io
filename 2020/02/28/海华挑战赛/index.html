海华挑战赛

Posted on 2020-02-28

This is my first time to attend this kind of competition. I’m certainly looking forword to it.

Brief introduction

This competition is built on the background of increasing garbage. The problem of how to deal with garbage classification efficiently is imminent. It is not realistic to solve this problem by human. Artificial intelligence must be used instead of human. to carry out efficient garbage classification.

Process

The main assignment of us is to design a program to recognize the images to effectively separate the rubbish.
We started by learning about convolution, which is part of understanding how to get a computer to recognize the edges of objects on a picture. Of course that alone is not enough for our computers to know how to recognize an object in its entirety. We also need to use color to help identify.
Even if it feels easy to say it now, it’s not so easy to actually do it. So we need to continue to practice and listen to the teacher to explain more about artificial intelligence knowledge.
