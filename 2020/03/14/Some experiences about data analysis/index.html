Some experiences about data analysis

After I learned some basic knowleges about Python, I start to learn numpy which is one impotant part of Python.
When I was first the book about numpy, I tought it is very similar to Python that I learned before. But these are only my initial impression of it. In fact, when I started to input his code on the website, I really felt that the Python he learned at the beginning was a little different, and his data processing would be more convenient, faster and easier for people to understand. This tool can be very useful when dealing with big data. I think if I can use this tool in combination with what I learned at the beginning, it will be very effective.
Of course, everything has advantages and disadvantages at the same time. It’s just that it’s so convenient that if you need to use numpy, your logic on writing programme has to be very clear. This means you need to spend more time reading the book to understand that this is different from the python logic you learned in the first place.

On top of that, you need to spend a lot of time writing code because once you’re familiar with the code. As a result, you’ll understand the numpy better. In class, the teacher would also encourage us to type the example code on the book and send screenshots in the group after writing the code. After class, we also have homework to practice on, which gives us an opportunity to write code.
